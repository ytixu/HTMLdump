Swapping Numbers

Write a function to swap a number in place without temporary variables.

One way to do this is to use arithmetics.

Ruby

def swap(a, b)
    a = a+b
    b = a-b
    a = a-b
    puts("#{a} #{b}")
end

But in binary, we can use another type of operation to obtain the same effect. Given two binary numbers $a$ and $b$, $a$ XOR $b$ is similar to $a+b$ up to the carry. That is, if $a=1$ and $b=0$, $a$ XOR $b = a+b = 1$; but if $a=1$ and $b=1$, $a$ XOR $b = 0$ while $a+b = 10$.

Because we discarded the carry, when we try to retrieve the original values we also need to omit borrows as in the substractions. To do so, we can also use XOR. Indeed, XOR is similar to addition and substraction. This property allows us to XOR two numbers, and then XOR the result with one of the original number to get back the other.

Ruby

def swapp(a, b)
    a = a^b
    b = a^b
    a = a^b
    puts("#{a} #{b}")
end

This property of XOR can be also useful for reversing strings.

Ruby

def reverse(str)
    $l = str.length
    for i in 0..$l/2 do
        str[i] = str[i]^str[$l-i-1]
        str[$l-i-1] = str[i]^str[$l-i-1]
        str[i] = str[i]^str[$l-i-1]
    end
    return str
end
