Find Missing Number

from Algorithms For Interviews

Given a file containing roughly 300 million social security numbers (9-digit numbers), find a 9-digit number that is not in the file. You have unlimited drive space but only 2MB of RAM at your disposal.

To find a missing number of 9 digits, we can create an array of all possible 9-digit numbers with value 0 or 1 indicating whether each number in the file as we read through it. But with 9 digits, there are 1 billion possible numbers, thus we need an array of 1 billion bits. That's not going to fit is our little RAM, which is only about 16 millions bits.
