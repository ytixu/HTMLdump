Multiples of 3 and 5

If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23.

Find the sum of all the multiples of 3 or 5 below 1000.

The sum of multiples of 3 below 1000 is \begin{equation*} \sum_{i=1}^{\lfloor 1000/3\rfloor} 3i = \sum_{i=1}^{333} 3i \end{equation*} Similarly, we get the sum of multiples of 5 below 1000 and append it to the previous equation. But some multiples of 5 are also multiples of 3, so we added multiples of 15 below 1000 twice.

The answer is thus \begin{equation*} 3\sum_{i=1}^{333} i + 5\sum_{i=1}^{199} i - 15\sum_{i=1}^{66} i = 233168 \end{equation*}

smlnj Code

fun answer (sum, n) = 
    if n = 1000 then
        sum
    else if n mod 3 = 0 orelse n mod 5 = 0 then
        answer(sum+n, n+1)
    else
        answer(sum, n+1);
        
answer(0,1);

haskell Code

answer n s
    | n == 1000 = s
    | mod n 3 == 0 = answer (n+1) (s+n)
    | mod n 5 == 0 = answer (n+1) (s+n)
    | otherwise = answer (n+1) s
    
main = print (answer 1 0)
