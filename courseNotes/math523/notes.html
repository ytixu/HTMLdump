Exponential Familiy

Consider $Y_i$ with distribution belonging to the linear exponential family (EF), \begin{equation} f_{Y_i}(Y_i|\theta) = \exp(-b(\theta)+ \theta y_i + c(y_i)) \end{equation} where $b(\theta)$ is a function of the parameters only and $c(y_i)$ is a function of the data only. Assuming that the first 2 partial derivatives exist for all $\theta$ in the define support of the paprameter space, let \begin{align} &I(\theta;y_i) = \log(f_{Y_i}(Y_i|\theta))\\ &\dot{I}(\theta;y_i) = \frac{\partial I(\theta;y_i)}{\partial\theta}\\ &\ddot{I}(\theta;y_i) = \frac{\partial^2 I(\theta;y_i)}{\partial\theta^2} \end{align} In particular \begin{equation} \dot{I}(\theta;y_i) = \frac{\partial}{\partial \theta} (-b(\theta)+ \theta y_i + c(y_i)) = -\dot{b}(\theta) + y_i \end{equation} Here are some interesting properties.
  1. \begin{equation} \label{eq:efexp} E\left[\dot{I}(\theta;y_i)\right] = 0 \end{equation}
  2. Let $E\left[m(y_i, \theta)\right] = 0$ for some function $m$, then \begin{equation} E\left[\frac{\partial m(y_i, \theta)}{\partial \theta'}\right] = -E\left[m(y_i, \theta)\frac{\partial I(\theta;y_i)}{\partial\theta'}\right] \end{equation}
  3. In the special case when $m(y_i, \theta) = \dot{I}(\theta;y_i)$, \begin{equation} E\left[\ddot{I}(\theta;y_i)\right] = -E\left[(\dot{I}(\theta;y_i))^2\right] = I(\theta;y_i) = Var(\theta, y_i) \end{equation}
