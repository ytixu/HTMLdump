To move your player (black tile):
Option 1: Use numpad numbers (1-9);
Option 2: Use arrow keys (cannot move in diagonal in this case);
Option 3: Use letter keys <Y>, <U>, <I>, <H>, <J>, <K>, <N>, <M>, <,>.

Use the same keys to highlight a tile next to the player.
To set flag on a highlighted tile, either use <Enter> (on numpad) or <Shift>.
To open a highlighted tile, either use <0> (on numpad) or <Space>.

When the surronding flags add up to the number on the tile, use <5>(on numpad) or <j> to open all surrounding non-flagged tiles.
